## E - Crested Ibis vs Monster

個数制限なしのナップザック問題。

```
DP[n][h] = 最初のn個を使ってAの和がhのときの最小コスト
DP[n][h] = min(DP[n-1][h], DP[n+1][h-A[n-1]]+B[n-1])
答えはDP[N][H]
```

として解けるが、python だと TLE となる。

```
DP[h] = 体力をh削るのに必要な魔力
DP[h+a] = min(DP[h+a], DP[h]+b)
答えはmin(DP[h]) (h>=H)
```

とすると、計算量は同じだが速くなる。\
ただし、処理そのままでは通らず、関数化すると通る。
